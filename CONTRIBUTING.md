## Purpose

This repository documents critical bugs in the Augment VS Code Extension v0.754.3. It is intended for:

1. **Augment Code team** — to understand and fix the reported bugs
2. **Security researchers** — to learn from the forensic analysis
3. **Affected users** — to understand the issues and apply workarounds

---

## How to Contribute

### Reporting Additional Evidence

If you've encountered these bugs and have additional evidence:

1. **Fork this repository**
2. **Add your evidence** to the `evidence/` directory
3. **Submit a pull request** with:
   - Description of your environment (OS, VS Code version, extension version)
   - Steps to reproduce
   - Logs or screenshots
   - Any additional findings

### Improving Documentation

If you find errors or have suggestions for improving the documentation:

1. **Open an issue** describing the problem
2. **Submit a pull request** with the fix

### Adding Test Cases

If you've discovered additional test cases that trigger these bugs:

1. **Add test script** to `reproduction/` directory
2. **Document the test** in `reproduction/README.md`
3. **Submit a pull request**

---

## Guidelines

### Evidence Quality

All evidence should be:
- **Reproducible** — others can verify your findings
- **Documented** — clear steps to reproduce
- **Attributed** — credit original discoverer if not you

### Code Quality

All code (test scripts, fix scripts) should be:
- **Well-commented** — explain what and why
- **Tested** — verify it works before submitting
- **Safe** — no destructive operations without warnings

### Documentation Quality

All documentation should be:
- **Clear** — easy to understand for non-experts
- **Accurate** — verified against actual code/behavior
- **Complete** — no missing steps or assumptions

---

## What NOT to Contribute

### Do NOT submit:

- **Proprietary code** from Augment extension (only reference line numbers)
- **Personal information** (email addresses, API keys, etc.)
- **Unverified claims** (speculation without evidence)
- **Duplicate reports** (check existing issues first)

## Code of Conduct

### Be Respectful

- This is a technical bug report, not a criticism of Augment team
- Focus on facts and evidence, not opinions or blame
- Assume good faith from all contributors

### Be Professional

- Use clear, professional language
- Provide constructive feedback
- Help others understand the issues

### Be Collaborative

- Share knowledge and findings
- Help verify others' evidence
- Work together to improve the report
